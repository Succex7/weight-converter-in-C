# Weight Converter

A simple **C program** that converts weight between **kilograms and pounds**.  
Users can choose the conversion direction and input their weight to get the result.

## Features
- Convert **kilograms to pounds**
- Convert **pounds to kilograms**
- Easy-to-use console interface
  
## Usage
1. Compile the program using a C compiler:
   ```bash
   gcc weight_converter.c -o weight_converter
2. Run the program: <br>
   ./weight_converter

3. Follow the on-screen prompts to select the conversion type and enter the weight.
